## Overview

AI Report Writer streamlines how educators generate student feedback:

- Collects student data once, then generates tailored reports with the click of a button.
- Lets teachers build reusable report templates, share them with colleagues, or browse the community library.
- Tracks monthly usage and remaining credits on the Usage Report page.
- Runs entirely in the browser; no installation required.

For the live app, visit [https://aireportwriter.app](https://aireportwriter.app).
